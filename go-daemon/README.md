# GoGet some stuff

Interactive Client and Daemon written in Go for interacting with a remote server.

## Installation

Installing the client and daemon is easy! Simply copy the following:

```sh
# build the binaries
go build -o myclient client/client.go
go build -o mydaemon daemon/daemon.go

# install the stuff
./install.sh
```

To uninstall, you can run the `uninstall.sh` script.
